# 2.3.1
## 新增
1. 历史记录支持预览缩略图
2. 历史记录支持批量选择与批量删除
3. 历史记录支持按类型、尺寸、彩色排序
4. 历史记录支持导出为 JSON / CSV 文件
5. 下载文件名支持自定义

## 优化
1. 统一三个页面的 CSS 颜色风格
2. 历史记录 Toast 提示样式与主页面保持一致
3. 历史记录改为双击行才放置丝印，单击仅显示提示
4. 导入图片页面反相切换时实时更新预览

## 修复
1. 修复保存历史记录时因 getExtensionAllUserConfigs 返回非对象导致保存失败的问题
2. 修复导入图片使用复杂多边形导致验证不通过及颜色异常的问题，改为始终使用内嵌二进制对象，反相由前端 Canvas 自行处理
